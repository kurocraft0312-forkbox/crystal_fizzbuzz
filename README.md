# FizzBuzz sample

Crystal - FizzBuzz sample

## Usage
```
$ fizzbuzz 10
1,2,Fizz,4,Buzz,Fizz,7,8,Fizz,Buzz
$ fizzbuzz 30
1,2,Fizz,4,Buzz,Fizz,7,8,Fizz,Buzz,11,Fizz,13,14,FizzBuzz,16,17,Fizz,19,Buzz,Fizz,22,23,Fizz,Buzz,26,Fizz,28,29,FizzBuzz
```
